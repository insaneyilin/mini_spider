# mini_spider

A multi-thread crawling spider demo

- mini_spider.py: 主程序 [x]
- crawl_thread.py: 实现抓取线程 [x]
- html_parser.py: 对抓取网页的解析 [x]
- downloader.py: 将网页保存到磁盘 [x]
- config_loader.py: 读取配置文件 [x]
- url_object.py: 表示 url 对象 [x]
- log.py: 日志相关 [x]
